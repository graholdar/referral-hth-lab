# referral-hth-lab

## 项目简介

`referral-hth-lab` 是一个用于归档与发布独立 HTML 页面的仓库。  
这些页面通常用于测试、演示、文档呈现或静态原型展示，不绑定任何特定域名或在线服务。  
本仓库旨在为开发者提供一个干净、通用的 HTML 文件存储空间，方便快速引用与版本管理。

## 目录说明

```
├── pages/          # 存放所有已归档的 HTML 页面
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── index.html      # 仓库首页（可选，用于展示页面列表）
└── README.md       # 本文件
```

- `pages/` 目录下每个 HTML 文件独立存在，文件名即页面标识。
- `assets/` 目录存放页面共享的静态资源，避免重复引入。
- 如需新增页面，直接在 `pages/` 目录添加 `.html` 文件即可。

## 页面归档说明

每个 HTML 页面均为自包含或引用公共资源的独立文件。  
归档原则：

- 页面内容保持稳定，不依赖外部在线 API 或动态服务。
- 文件名应反映页面用途或版本号，便于识别。
- 如需更新页面，建议保留旧版本并添加新文件，或通过提交记录说明变更。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或修正已有页面中的问题。
- 请确保新增页面不包含敏感信息、外部追踪代码或恶意内容。
- 若页面依赖特定外部资源，请在文件头部注释中说明。
- 本仓库仅作静态托管用途，不提供后端支持。
